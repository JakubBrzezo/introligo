# User Service

The User Service is the authentication and authorization hub of the platform. It manages user accounts with role-based access control supporting Admin, User, and Moderator roles.

## Key Capabilities

- **User Management** - Full CRUD operations for user accounts
- **Role-Based Access** - Three-tier role system (User, Admin, Moderator)
- **Audit Trail** - Timestamp tracking for all user operations
- **User Preferences** - Customizable settings per user
